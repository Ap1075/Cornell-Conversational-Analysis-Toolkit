Question Typology
==================

Implements unsupervised identification of rhetorical roles in questions, see
as described in this `paper <http://www.cs.cornell.edu/~cristian/Asking_too_much.html)>`_.

Example usage: `extracting common question types in UK parliament <https://github.com/CornellNLP/Cornell-Conversational-Analysis-Toolkit/blob/master/examples/question-typology/parliament_questions_example.ipynb>`_
